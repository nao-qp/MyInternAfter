///TomCatインストール
（jarファイルでデプロイする場合はSpringBootごとデプロイするので不要。
本番環境の場合はwarファイルなのでTomcatが必要）

移動
cd /opt 

ダウンロードして解凍する方法とaptでインストールする方法がある
今回はapt

Ubuntuのパッケージリストを更新 まず、最新のパッケージリストを取得します。
sudo apt update

Tomcatのインストール apt で Tomcat をインストールする
sudo apt install tomcat10

インストールの確認 Tomcat が正しくインストールされたか確認するために、
Tomcat サービスが起動しているかチェックします。
sudo systemctl status tomcat10
起動中active (running)

接続確認
IPアドレス確認
ip a
結果
 172.29.197.138
http://172.29.197.138:8080
tomcatのページが表示されればOK

javaがインストールされている確認
javac -version
結果
javac 21.0.5

停止
sudo systemctl stop tomcat9

TomCatユーザーを作成
sudo useradd -r -m -U -d /usr/share/tomcat10 -s /bin/false tomcat
